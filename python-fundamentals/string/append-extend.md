# append, extend

## append:

```python
alist1 = ["apple", "banana", "orange"]
alist2 = ["pear", "peach", "watermelon"]
alist1.append(alist2)
print(alist1)

#['apple', 'banana', 'orange', ['pear', 'peach', 'watermelon']]
```

## extendï¼š

```python
alist1 = ["apple", "banana", "orange"]
alist2 = ["pear", "peach", "watermelon"]
alist1.extend(alist2)
print(alist1)

#['apple', 'banana', 'orange', 'pear', 'peach', 'watermelon']
```
