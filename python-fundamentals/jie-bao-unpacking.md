# 解包 Unpacking

解包就是将容器里面的元素逐个取出来

### 元组解包

```python
>>> a,b,c = (1,2,3)
>>> a
1
>>> b
2
>>> c
3
```

### 字符串解包

```python
>>> a,b,c = "abc"
>>> a
'a'
>>> b
'b'
>>> c
'c'
```

### 字典解包

```python
>>> a,b,c = {"a":1, "b":2, "c":3}
>>> a
'a'
>>> b
'b'
>>> c
'c'
```

字典解包后，只会把字典的 key 取出来，value 则丢掉了。

在某个变量面前加一个星号，而且这个星号可以放在任意变量，每个变量都分配一个元素后，剩下的元素都分配给这个带星号的变量

```python
>>> a, *b, c = [1,2,3,4]
>>> a
1
>>> b
[2, 3]
>>> c
4
```
