# 常见问题

### **Appium可以做什么？**

* 安卓app自动化
* ios app 自动化
* web 应用自动化
* Windows 桌面应用自动化

### Appium-android 环境搭建需要准备什么?

Python-appium-client 库，JDK, AndroidSDK, Appium-server, 手机真机

### 自动化手机模式的页面有哪些方法？

* Selenium 电脑端模拟手机模式页面
* Appium 自动化混合app的web页面部分
* 测试html源码

### 打开通知栏是哪个方法?

* 模拟向下滑动屏幕操作
* 调用 `driver.open_notification()` 方法
